# Windows Repair Tool

A comprehensive all-in-one Windows troubleshooting and repair utility that automates many of the most common maintenance and recovery tasks through an easy-to-use interactive menu.

> **⚠️ IMPORTANT:** This script **must be run as Administrator**. Right-click the `.bat` file and select **Run as administrator**. Administrative privileges are required to repair system files, manage Windows services, reset networking, create restore points, and perform other system-level operations.

## Features

* Full Windows repair sequence (recommended)
* DISM & SFC system file repair
* Windows Update reset and repair
* Network and Internet reset
* Windows Security and Microsoft Store repair
* Disk health checks and component cleanup
* Firewall reset
* Temporary file cleanup
* Printer Spooler repair
* Windows Search index rebuild
* Icon and thumbnail cache rebuild
* Time synchronization
* Battery, power, and system information reports
* Windows Memory Diagnostic launcher
* Detailed logging of every operation

## Safety

* Creates a System Restore Point before major repairs (when available)
* Does **not** delete personal files or installed applications
* Uses only built-in Microsoft Windows tools and commands
* Saves a detailed repair log to your Desktop

## Supported Operating Systems

* Windows 10
* Windows 11

For the best results, restart your computer after running major repairs.
